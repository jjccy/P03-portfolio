Project overview

Project Description

Image Mood Converter is a one month course project for IAT 455: Computational Media. The goal for this project is to explore the use of color in changing the emotional and mood of a particular image. Through this, we hope to achieve a level of modification such that the supplied images, if they are rather gloomy, with grey skies or generally less appealing colors, can be modified to be warmer and brighter.

We notice that this can be done in Photoshop or other image editing programs, but the problem with those is that the emotional/mood color shift of those programs require the manual brushing from the user. In this program, we wish to automate it so that the user can color shift many images in a short period of time.

Process

Inspiration

After we decide to the general goal for this project, I start to do research on these area. One research paper really inspired me is called Automatic Mood-Transferring between Color Images, written by Yang, Chuan-kai and Peng, Li-Kai. This paper classified an image into 24 color moods where each image has a different percentage of moods. The paper also introduces an algorithm to shift input image’s mood to the reference image’s mood by recalculating the RGB value.

We plan to create a program with a GUI on the right side that allows users to upload their own images (first through direct upload and later possibly by grabbing images from the internet URL itself). After successful upload, the user can click a number of buttons, each with its own emotional or mood colors where the image’s colors will shift accordingly.
